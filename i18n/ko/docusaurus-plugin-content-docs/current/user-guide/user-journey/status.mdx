# 유저 여정 상태와 수정

## 유저 여정 상태

유저 여정은 비활성/일시 중지/활성 상태를 가집니다. 활성 상태인 유저 여정만 새로운 진입과 메시지 발송등의 시나리오가 동작합니다.
유저 여정을 최초로 생성한 시점에는 비활성 상태의 유저 여정이 만들어지며, 활성 상태로 변경할 수 있습니다.
일시 중지 상태의 유저 여정은 새로운 진입과 노드 실행이 일시정지되며 다시 활성화시 대기중인 세션이 계속해서 진행됩니다.

다만, 정확한 분석과 안정적인 발송을 위해서는 일시 중지가 아닌 새로운 유저 여정으로 시작하는 것을 추천드립니다.

#### 유저 여정 상태

| 상태     | 설명                                                                                                     |
| -------- | -------------------------------------------------------------------------------------------------------- |
| 활성     | 진입 조건에 따른 새로운 세션 진입과 진입한 세션의 노드 실행이 정상적으로 수행됩니다.                     |
| 일시중지 | 새로운 세션이 시작되지 않으며, 진행중인 세션이 있다면 노드를 실행하지 않고 대기합니다.                   |
| 비활성   | 유저 여정이 비활성 상태로 전환되면 진행중이던 모든 세션은 이탈 처리되며 새로운 세션이 시작되지 않습니다. |
