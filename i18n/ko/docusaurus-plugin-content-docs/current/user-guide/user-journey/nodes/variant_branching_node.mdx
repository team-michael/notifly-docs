# 랜덤 분기 노드

이 노드에 도달한 유저를 일정 비율에 맞춰 무작위로 하위 노드에 할당합니다. 이 노드에 도달하면, 하위 노드로 즉시 분기됩니다.

## 랜덤 분기 설정

랜덤 분기 노드는 각 노드로 할당할 비율을 설정할 수 있으며, 모든 분기 노드의 비율의 합은 100%가 되어야 합니다. 노드는 최대 5개까지 설정할 수 있습니다.
이 노드에 도달한 유저는 재진입하더라도 동일한 분기 노드에서는 동일한 결과로 분기됩니다. 또한, 각 분기 노드별 지표를 분석할 수 있습니다. 다만 AB 테스트를 위한 검정값이나 통계를 제공하지 않습니다.
