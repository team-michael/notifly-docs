# 카페24 연동

카페24로 쇼핑몰을 이용하시는 경우 개발 없이 바로 노티플라이를 설치하고 마케팅을 진행하실 수 있습니다.  
노티플라이-카페24 앱을 설치하여 카페24 데이터를 실시간으로 연결하고 자동화 메시징 캠페인을 실행하세요.

## 노티플라이 설치하기

1. 노티플라이와 사전 계약을 진행해주세요!
   카페24 앱스토어에서 [문의하기]를 눌러 사전 계약 진행 희망의사를 밝혀주시면, 계약 및 노티플라이 계정 생성을 도와드립니다.

2. 카페24 앱스토어에서 [설치하기]를 눌러주세요.

<img
  src={require("./img/cafe24-plugin-install-start-button.png").default}
  width="900px"
/>

3. 접근 권한에 동의하면, 카페24 자사몰과 노티플라이의 연동이 시작됩니다.

<img src={require("./img/cafe24-plugin-agreement.png").default} width="900px" />

4. 연동 작업 시작 안내가 뜨면 [계속하기] 를 눌러주세요.

<img
  src={require("./img/cafe24-plugin-install-process.png").default}
  width="900px"
/>

5. 연동할 프로덕트를 선택해주세요.

<img
  src={require("./img/cafe24-plugin-select-product.png").default}
  width="900px"
/>

6. 다음으로 이 화면이 떴다면, 연동작업이 완료된 것입니다.  
   [노티플라이 유저 리스트 바로가기] 버튼을 누르면 자사몰 회원 정보가 노티플라이와 연동된 것을 확인하실 수 있습니다. 연동된 회원정보를 활용하여 고객에게 메시지를 발송해보세요.

<img
  src={
    require("./img/cafe24-plugin-install-complete-goto-user-list.png").default
  }
  width="900px"
/>
